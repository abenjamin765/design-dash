# /orca-teach

Loads `skills/3-object-modeling/ooux-primer/SKILL.md`.

Teaches OOUX concepts — the Four Ancient Truths, the ORCA process, and how to apply them — tailored to the learner's role and experience level.

## Usage

```
/orca-teach
```

## What it does

1. Loads `skills/3-object-modeling/ooux-primer/SKILL.md`
2. Asks about the learner's role, prior UX experience, and what they want to understand
3. Delivers an interactive, example-driven explanation of OOUX/ORCA concepts at the right level
4. Optionally transitions into a hands-on ORCA exercise

## Related commands

- `/orca-start` — apply the concepts to a real project
- `/design-dash` — full orchestrated workflow
