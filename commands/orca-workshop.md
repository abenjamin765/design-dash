# /orca-workshop

Loads `skills/0-orchestration/facilitation-kit/SKILL.md`.

Generates workshop agendas, pre-read packets, icebreakers, timebox templates, and facilitation guides for live or async OOUX/ORCA sessions with cross-functional teams.

## Usage

```
/orca-workshop
```

## What it does

1. Loads `skills/0-orchestration/facilitation-kit/SKILL.md`
2. Interviews you about session goals, participants, and time constraints
3. Generates a complete facilitation pack: agenda, pre-read, activities, timers, debrief prompts

## Related commands

- `/design-dash` — full orchestrated dash
- `/orca-start` — planning and intake
- `/orca-panel` — adversarial critique session
