# /orca-start

Loads `skills/1-intake/orca-planner/SKILL.md`.

Builds a sequenced ORCA workflow plan tailored to your project and guides you through executing each skill step by step. Use this when you want a structured intake session that produces a prioritized skill sequence before committing to a full dash.

## Usage

```
/orca-start
```

## What it does

1. Loads `skills/1-intake/orca-planner/SKILL.md`
2. Interviews you about project context, constraints, and goals
3. Recommends a sequenced skill plan (which ORCA steps, which order, which gates)
4. Optionally transitions directly into execution

## Related commands

- `/design-dash` — full nine-phase orchestrated workflow
- `/orca-workshop` — workshop facilitation kit
